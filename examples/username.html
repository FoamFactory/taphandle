Username Field Examples

Note

For all examples, the prefix used is ninkasi.

The following is an example of how to use Taphandle with a username field for validation. A username field is defined as an <input> tag with type="username".

The following is an example of how to skip auto-validation of the entries, if, for example, you want to always show an error message, want your own validation code to run on a specific element, or simply don't want the validation to happen. You can achieve this through the data-skipautovalidation attribute on the form field.
