Password Field Examples

Note

For all examples, the prefix used is ninkasi.

The following is an example of how to use Taphandle with a password field for validation. A password field is defined as an <input> tag with type="password". For both "password" and "password confirmation" inputs, you will need to specify a specific class to distinguish them. The "password" field should have a type of password and a class including PREFIX_passwordField. The "password confirmation" field should have a type of password and a class including PREFIX_passwordConfirmationField.
